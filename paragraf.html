Web adalah layanan online yang bisa diakses menggunakan internet. Web terdiri dari beberapa jenis seperti web lokal dan web hosting.

Web berjalan dengan konsep client-server. Client yang dimaksud adalah web browser seperti chrome, mozila dan IE. Sedangkan Server yang dimaksud seperti Apche atau yang lain.

Pada sisi client, web browser mengubah kode html menjadi tampilan yang menarik dengan sentuhan CSS. Selain itu web akan menjadi lebih responsif dengan adanya kode javascript.
